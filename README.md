# LyricLabel

## The Pitch

LyricLabel is an async MP3 metadata CLI that uses Last.fm with resilient filename parsing and retry-aware fetching to safely enrich your music library at scale.

## Quick Start

```bash
uv tool install .
lyriclabel path/to/music/ --dry-run
```
